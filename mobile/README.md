# Mobile (Flutter)

This folder is reserved for the Flutter mobile app.

To initialize a new Flutter project here, run from the repo root:

```bash
flutter create . --project-name audio_pipeline_mobile
```

Or create a new Flutter app in a subfolder:

```bash
cd mobile
flutter create .
```

Then connect the app to the server API (see `server/` and root `README.md` for API docs).
